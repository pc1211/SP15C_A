Dashboard

To create a new Chrono/Timer (CT), Press (Chrono) or (Timer).
If is ON, the new CT is directly added to the list.
If is OFF, the Single Chrono/Timer screen is called to show only the new CT, which will appear in the list on return.
Each CT in the list shows its row of current state/command buttons:

  • Select (checkbox)
  • Mode ( or) and running state
  • Split ()
  • Alarm set in Clock app ()
  • Time
    • if chrono: elapsed time
    • if timer: remaining time
      (or expiration time if is ON)
  • Label
CT are sorted by order of creation.
