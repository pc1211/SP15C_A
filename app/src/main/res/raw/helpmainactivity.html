SP15C

Main differences:

  • No Complex mode nor MATRIX operations
  • 10 digits displayed (Calculations made with Java Double type, up to 10^307, 0^0 = 1)
  • Alphanumeric error messages
  • Up to 1000 registers (default: 100)
  • Up to 10000 program lines (with alphanumeric symbols (includes TEST and HYP functions) or key codes (pressing the display))
  • CLEAR PRGM in EDIT mode aks for confirmation
  • PSE while running program displays X for about 1 second, without stopping the program
  • GSB deepest level: 100
  • All flags are user flags (no system flags)
  • SOLVE implements Secant method
  • INTEG implements Romberg method
  • STO RAN#, Re<>im, MATRIX, RESULT, MEM (useless) are inactive
